📘 Programming Pathshala — Figma Webpage Clone

A responsive landing page built using HTML5 and CSS3, created from a Figma design layout.
This project represents my first complete UI implementation from design to code.

------------------------------------------------------------------------------------------------------
🚀 Project Overview

This project is a modern educational landing page inspired by a Figma design for Programming Pathshala. The goal was to practice:

Converting Figma UI designs into real webpages

Writing clean semantic HTML

Creating responsive layouts using CSS

Building reusable UI components

Implementing mobile responsiveness using media queries

The website includes a hero section, learning roadmap, CTA section, and footer — similar to real-world EdTech landing pages.

------------------------------------------------------------------------------------------------------

📱 Responsive Design

The webpage adapts to smaller screens using:

Flexbox direction changes

Grid restructuring

Hidden navigation links on mobile

Stack-based layout for sections

Optimized spacing and typography

Breakpoint used:
@media screen and (max-width: 768px)
